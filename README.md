## ENVS

To run localy you must create a .env in the root path with the name `.env-developtment` or add the variables to the environment values of your system

````
APP_NAME=taste-match-api
GO_ENV=development

DB_ENABLE_LOG=true
DB_HOST=
DB_NAME=
DB_PORT=
DB_TYPE=mysql or postgresql
DB_PASS=
DB_PASS_MIGRATION=
DB_USER=
DB_USER_MIGRATION=

SERVER_BASE_PATH=v1
SERVER_PORT=8000
````
